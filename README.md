# Connect 4

## Compile

### Linux

```
git clone https://github.com/Escapehub/Connect4.git
cd connect4
mkdir build 
cd build
cmake ..
make
```

### Windows

```
git clone https://github.com/Escapehub/Connect4.git
cd connect4
wget https://www.sfml-dev.org/files/SFML-2.5.1-windows-vc15-64-bit.zip
unzip SFML-2.5.1-windows-vc15-64-bit.zip
mkdir build
cd build
cmake ..
```
